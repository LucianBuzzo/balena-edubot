# Balena + CamJam + Edublocks
 
- Build your robot with the [CamJam EduKit 3](https://camjam.me/?page_id=1035)
- Setup and manage the device with [balena cloud](https://dashboard.balena-cloud.com/)
- Program your robot with the [Edublocks](https://app.edublocks.org/) interface
